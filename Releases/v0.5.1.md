# UTools 0.5.1

## Release Notes

- Refined the repository into a hybrid layout: the repo remains a normal Unity project, while `Assets/UTools` is exposed as the installable package root for Git URL usage.
- Restored example scenes, assets, and scripts to `Assets/UTools/Example` so the in-repo demo workflow continues to work without broken references.
- Kept package metadata in `Assets/UTools/package.json`, allowing users to install UTools directly with `?path=/Assets/UTools`.
- Clarified README installation instructions for Git URL / UPM usage and documented the project-plus-package structure more explicitly.
- Preserved the official `TextMeshPro` package dependency rather than introducing extra third-party runtime dependencies.
