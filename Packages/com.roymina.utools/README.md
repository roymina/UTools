# UTools

UTools is a lightweight Unity toolkit for dependency injection, messaging, object lookup, and utility helpers.

## Install via Git URL

Add the package with:

`https://github.com/roymina/UTools.git?path=/Packages/com.roymina.utools`

## Notes

- `com.unity.textmeshpro` is declared as a package dependency for sample UI.
- If imported samples show missing TMP assets, run `Window > TextMeshPro > Import TMP Essential Resources` once.
- Runtime code lives under `Scripts/`.
- Tests live under `Tests/`.
- Sample assets live under `Samples~/`.
